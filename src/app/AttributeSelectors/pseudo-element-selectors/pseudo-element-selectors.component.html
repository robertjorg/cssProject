Pseudo Element Selectors

Styling the First Letter

You can set the size of the first letter by doing .first-letter::first-letter {{'{'}} color: red;{{'}'}}

You can also style the first line
Does it work with a break?
This is done by p.first-line::first-line {{'{'}} color: purple; font-size: 300% {{'}'}}

Before or after elements

You can do this by doing body::after {{'{'}} content: "The End.";{{'}'}}
