Pseudo-class Selectors

Selectors that let you assign styles to phandom classes that are inferred by the state of elements, markup patterns in the document, or the state of the document.

  • Selecting the root element :root - select the root of an element
  • Selecting empty elements :empty - select any element that has no children of any kind, including text nodes. p:empty {{'{'}} display: none;{{'}'}} would prevent empty paragraphs from showing (must have no whitepace or text of any sort).
  • Selecting unique children :only-child - select elemnt when they are the only child element of another element. a[href] img:only-child {{'{'}}border: 2px solid black;{{'}'}}
  • Selecting only types of :only-of-type - will match any element that is the only of its type among all its siblings.
  • Selecting first children :first-child - used to first children of an element.
  • Selecting last children :last-child - just like above
  • Selecting first of a type :first-of-type - select the first item of a given type, within an element. Not the entire document
  • Selecting last of a type :last-of-type - selects the last item of a given type, within an element. Just like first!
  • Selecting every nth of a type :nth-child(n) - selects the nth child of a given type. Allow you to do every other or whatever nth you choose by doing xn (or xn + y)
  • Selecting the last child and work backwards :nth-last-child()
  • There is also :nth-of-type() and :nth-last-oftype() - works similar to the two above

p:first-child {{'{'}}font-weight: bold;{{'}'}}
li:first-child {{'{'}}text-transform: uppercase;{{'}'}}
The p:first-child will make the first p that is a child of an element be bold faced.
The li:first-child will make the first li element that is a child of an element be uppercase.
Remember that the :first-child does not select the first child of an element but acts as a phantom class where your element is the first child of a class.

Last child is the mirror of first-child. It works the same, except using the last child instead of the first child.
