Universal Selector

The universal selector does not contribute to specificity.
An example is * {{'{'}} color: gray; {{'}'}} /* 0,0,0,0 */
The universal selector * has no specificity. If a selector has a universal selector in it, the specificity does not change:
  • div p /* 0,0,0,2 */
  • body * strong /* 0,0,0,2 */
