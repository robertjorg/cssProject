Weight and Origin

If two rules apply to an element and one has !important marked, the one with !important will win. Even if there is a style in html element !important will win out. For example:
  • p {{'{'}}color: gray !important;{{'}'}}
  • {{'<'}}p style="color: black;"{{'>'}}Well, {{'<'}}em{{'>'}}hello{{'<'}}/em{{'>'}} there!{{'<'}}/p{{'>'}}
The first item in the list wins due to the important. But if you add important to the inline stayle it will win.

When the weight of the rule is the same, the origin of a rule is considered. The author's style sheet will win over the reader's style sheet. Unless there is an !important on both. Then the reader's style sheet will win.
